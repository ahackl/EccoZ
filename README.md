# EccoZ

Manage and collect meter readings for natural gas, water, electricity and many more


## Technology

EccoZ use:

UI
* [ioninc]
* [patternLock]

Database
* [pouchdb]

Export of data
* [webdav-client]

Charts
* [angular-chart]
* [C3]
* [D3]

Translation
* [angular-translate]

Tools
* [Underscore]
* [jQuery]


## Usage

Install the ionic framework and copy the files to the www directory.
