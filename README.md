# EccoZ

Manage and collect meter readings for natural gas, water, electricity and many more


## Technology

EccoZ use:

UI
* [ionic]
* [patternLock]

Database
* [pouchdb]
* [angular-pouchdb]


Export of data
* [webdav]
* [email]

Charts
* [angular-chart]
* [C3]
* [D3]

Translation
* [angular-translate]

Tools
* [Underscore]
* [jQuery]
* [Moment]
* [RainbowVis-JS]

## Usage

Install the ionic framework and copy the files to the www directory.
